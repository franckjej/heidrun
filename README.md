# Heidrun

A Mac Hotline client. Swift 6 port and modernisation of the 2002 original
by Göran Granström.

## Install

Drag **Heidrun** into your **Applications** folder. Launch from Spotlight.

Requires **macOS 15 (Sequoia)** or later. Notarized for Apple Silicon
and Intel.

See `CHANGELOG.md` for what shipped in this release.

## Heritage

The original Heidrun was written by **Göran Granström** ("GG") in 2002
as a modular Hotline client for Mac OS X. This Swift 6 port
reimplements the same wire protocol and the same broad feature shape,
modernised for macOS 15+. The original's plug-in modules were
GPL-2.0; this port honours that licensing for the whole project.

See `NOTICE.md` for the full credit + licensing rationale, and an
open invitation to GG to clarify if he'd like the licensing of this
port changed.
